# 🎥 Summarize EduVideo

A Python project that automatically generates concise summaries from YouTube educational videos.

## Features
- Extracts transcript from YouTube
- Cleans transcript text
- Uses Hugging Face Transformers for summarization
- Generates readable summaries
- Implemented in Kaggle Notebook

## Technologies
- Python
- Hugging Face Transformers
- youtube-transcript-api
- Kaggle
- PyTorch

## Workflow

YouTube Video
      ↓
Extract Transcript
      ↓
Preprocess Text
      ↓
Hugging Face Model
      ↓
Generate Summary

### Input
Long transcript from a YouTube lecture...

### Output
A concise educational summary.
